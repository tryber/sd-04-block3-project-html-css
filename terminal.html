Terminal

Terminal application

There are a lot of terminal applications around there, and I have two applications that I most like depending on the operating system.

Linux

When it comes to Linux, my favorite terminal is Konsole. If you already used KDE Plasma desktop envirnment, you probably are familiar with it because it is the default terminal on this DE.

Konsole has a good amount of settings and personalization, letting you define shortcuts, default shell, theme, font, transparency, and have integration with other KDE applications such as Dolphin file manager, which is perfect to make your terminal consistent with your system.

Windows

When it comes to Windows, I don't like PowerShell or Command Prompt for development purposes, which makes Hyper my favorite terminal, it is open-source and is made with web technologies including JavaScript, and like Konsole, Hyper has a lot o possible cutomizations including theme, font, transparency and default shell. All the configuration is done using the own shell or through a .js settings file. In the official website there is a page just for themes, and they are pretty easy to install.

Shell

The default shell in most Unix systems is Bash, and it is a really good Shell, but my favorite one is Zsh which is the default Shell on macOS Catalina. The main reasons I use Zsh instead of Bash is command-line completion and the possibility to use themes. But another good reason for using Zsh, is a framework called Oh My Zsh.

Oh My Zsh

Oh My Zsh is a framework for Zsh which is open-source and community-driven and it helps you manage your Zsh configuration and comes with a lot of helpers functions, plugins and themes. My favorite part are the plugins.

Plugins

There are two ways to install plugins on Oh My Zsh, manually or using a plugin manager. I prefer to use a plugin manager called Zinit, which automatically downloads and install all plugins described on the .zshrc file.


The plugins that I use are:


  1. fast-syntax-highlighting - For rich syntax highlighting
  2. zsh-autosuggestions - Gives suggestions based on history and completions
  3. zsh-completions - Gives completions scripts not available on Zsh yet

Theme

Like mentioned before, it is possible to install themes on Zsh, and the theme that I use is called Spaceship, which is a minimalistic theme that have really good features that you can see on the official GitHub page.


I have the following configurations on the .zshrc file:


      
        SPACESHIP_PROMPT_ORDER=(
          user          # Username section
          dir           # Current directory section
          host          # Hostname section
          git           # Git section (git_branch + git_status)
          hg            # Mercurial section (hg_branch  + hg_status)
          exec_time     # Execution time
          line_sep      # Line break
          vi_mode       # Vi-mode indicator
          jobs          # Background jobs indicator
          exit_code     # Exit code section
          char          # Prompt character
        )

        SPACESHIP_PROMPT_ADD_NEWLINE=false
        SPACESHIP_CHAR_SYMBOL="‚ùØ"
        SPACESHIP_CHAR_SUFFIX=" "
