5 Extensões que todo programador deveria baixar

Ter diversas ferramentas é muito importante principalmente para quem busca agilidade e qualidade no desenvolvimento de projetos. Existem muitas extensões gratuitas oferecidas pelo Google Chrome que vão atender às suas demandas e economizar muito o seu tempo.
Neste artigo, listei 5 extensões específicas para o navegador mais popular na atualidade, o Google Chrome, mas que podem ser facilmente encontradas em outros navegadores modernos. Elas vão salvar seu dia e melhorar muito seu fluxo de trabalho com relação à front-end, performance, mobile, acessibilidade, design, entre outras coisas.

  1. Web Developer

    É uma extensão bem completa para quem gosta de analisar praticamente tudo de sites (próprios e dos outros). Ele possui mais de 100 funcionalidades distribuídas entre os elementos principais de sites: CSS, imagens, formulários, tamanhos de telas, tools entre outros.

  2. CSS Peeper

    Outra extensão fantástica é o CSS Peeper. O design simples dele já impressiona quando você abre a extensão, informando para você o tamanho do arquivo CSS e o tempo de carregamento.

  3. Window Resizer

    Window Resizer emula diferentes tamanhos de janela, mostrando como o seu site pode ser visto por diferentes resoluções. Desde as pequenas telas dos celulares até aos monitores enormes que existem atualmente, esta extensão permite que você simule qualquer tamanho de janela, para saber se o seu site está sendo visto corretamente ou se existe algo que você possa fazer para melhorar a experiência das pessoas que o acessam.

  4. colorzilla

    É uma extensão simples mas essencial para designers e web designers. O recurso principal é o seletor de cores, que pode pegar uma cor em qualquer página ou elemento da Web e dar-lhe o código RGB / Hex / etc.

  5. Wappalyzer

    Esta ferramenta mostra qual foi a tecnologia utilizada para criar um site. Ele detecta sistemas de gerenciamento de conteúdo, plataformas de E-comerce, estruturas da web, software de servidor e ferramentas de análise.
