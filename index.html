List and View

Software de listagem e visualização de produtos

Produto Código Valor
  • Playstation 4
    		•  #45965 R$ 2.350,00
  • Processador Core I7
    		•  #442233 R$ 1.700,00
  • Exemplo
    		•  #00000 R$ 0,00

    Output

    1. Playstation 4 | #45965 | R$ 2.350,00 | >> 25-03-2020- 21:29
    2. Playstation 4 | #45965 | R$ 2.350,00 | >> 25-03-2020- 23:00
    3. Processador Core I7| #442233 | R$ 1.700,00 | >> 25-03-2020- 23:29

    O programa ira inserir os produtos selecionados no quadro de Output
    onde podera ser exportada em formato *TXT, *CLS ou *PDF.

    O usuario tambem podera inserir valor por meio de linha de comando.
    (ex. listview insert < nome do produto >)

    (ex. listview /?) << para obter uma lista de comandos.
